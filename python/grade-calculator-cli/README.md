# Grade Calculator CLI

## Description
A Python command-line application that calculates the average of multiple subject grades and provides performance status or remarks.

## Features
- Accepts multiple subject grades
- Validates user input (numbers only, 0–100 range)
- Calculates average automatically
- Displays performance status:
    - **Excellent** (90+)
    - **Good** (75–89)
    - **Needs Improvement** (<75)

## Concepts Used
- Loops (`while`)
- Conditional Statements (`if-elif-else`)
- Input validation 
- Exception handling (`try-except`)

## How to Run
Run the following command in your terminal:
```bash
python main.py
```

## Sample Output 
```bash
SIMPLE GRADE CALCULATOR

How many subjects: 3
Enter grade 1: 85
Enter grade 2: 90
Enter grade 3: 80

--- RESULTS ---
Average: 85.00
STATUS: GOOD
```
